# 💧 Cap Squeeze

> Squeeze the Flavor. Love the Water.

Cap Squeeze is a flavored bottle cap hydration brand — a simple cap that turns any standard water bottle into a personalized flavored drink experience. Zero sugar, all natural, and compatible with most 28mm standard bottles.

## 🚀 Getting Started

This is a static HTML/CSS/JS website — no build tools required.

**To run locally:**
```bash
# Option 1: Open directly
open index.html

# Option 2: Use a local server (recommended)
npx serve .
# or
python3 -m http.server 8080
```

Then visit `http://localhost:8080`
